## 🚀 Featured Project: AnalogCheck — AI-Powered SPICE Error Checker

**Python · ChromaDB · Sentence Transformers · LLM (OpenRouter) · ngspice**

An engineering copilot that catches semantic errors in SPICE netlists that simulators pass silently. Combines deterministic rule-checking with a Retrieval-Augmented Generation (RAG) pipeline for AI-driven diagnosis.

> **Perfectly aligned with AI Catalyst / Engineering Copilot roles.** RAG pipeline, LLM orchestration, domain-specific knowledge base, CLI tooling.
