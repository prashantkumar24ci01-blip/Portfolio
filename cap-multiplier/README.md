# CMOS-Based Grounded Capacitance Multiplier using PFTFN

**Technologies:** OrCAD PSpice, 0.35µm CMOS
**Timeline:** 2025

## What it does

Designed a 14-transistor cascode OTA-based capacitance multiplier achieving 1.503 nF equivalent capacitance with 0.2% accuracy using a PFTFN (Positive Four-Terminal Floating Nullor) topology.

## Key results

| Parameter | Value |
|-----------|-------|
| Equivalent capacitance | 1.503 nF |
| Accuracy | 0.2% |
| Technology | 0.35 µm CMOS |
| Transistor count | 14 |

## Verification

- **Monte Carlo analysis** — 1000-run statistical variation
- **PVT corners** — SS (slow-slow), TT (typical-typical), FF (fast-fast)
- **Temperature range** — -40°C to +85°C
