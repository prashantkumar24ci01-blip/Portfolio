# VCII-Based Simulated Grounded Inductor

**Technologies:** Cadence Virtuoso, 0.18µm CMOS, DRC/LVS/PEX
**Timeline:** 2025

## What it does

Designed custom analog layout of a 12-transistor VCII (Second Generation Voltage Conveyor) circuit configured as a simulated grounded inductor. Full Cadence flow: schematic → DRC/LVS → PEX → post-layout simulation.

## Performance

| Parameter | Value |
|-----------|-------|
| Equivalent inductance | 2.5 mH |
| Bandwidth | 50 kHz – 2.5 MHz |
| Power | 0.65 mW |
| Technology | 0.18 µm CMOS |

## Key skills demonstrated

- Cadence Virtuoso schematic entry
- Custom analog layout (matching, guard rings, symmetry)
- DRC (Design Rule Check) — clean
- LVS (Layout vs. Schematic) — verified
- PEX (Parasitic Extraction) — completed
- Post-layout simulation — validated
